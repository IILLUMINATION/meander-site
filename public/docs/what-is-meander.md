# Создание пустого квеста

> Добро пожаловать в Meander — инструмент для создания текстовых квестов с визуальным редактором.

## Создание квеста

### Шаг 1. Откройте форму создания

Запустите Meander. На главном экране перейдите во вкладку **«Библиотека»** и нажмите кнопку **«Создать квест»** (иконка `+`).

### Шаг 2. Заполните форму

| Поле | Обязательно | По умолчанию | Описание |
|------|:-----------:|:------------:|----------|
| **Название** | ✅ | — | Имя квеста, отображается в библиотеке |
| **Автор** | ❌ | — | Ваше имя или псевдоним |
| **Описание** | ❌ | — | Краткое описание сюжета |
| **Версия квеста** | ❌ | `1.0.0` | Версия вашего проекта |

Для создания достаточно ввести только **название**. Остальные поля можно заполнить позже.

### Шаг 3. Нажмите «Создать»

Meander создаст пустой проект и откроет **экран деталей квеста** — отсюда начинается вся работа над квестом.

---

## Что дальше

Пустой квест — это чистый холст. Следующие шаги:

1. **Создать главы** — разделы сценария (раздел «Сценарий и Главы»)
2. **Добавить ноды** — блоки с текстом и выборами (канвас-редактор)
3. **Настроить стартовую ноду** — точка начала игры

Как устроен квест внутри — читайте в следующей статье.
